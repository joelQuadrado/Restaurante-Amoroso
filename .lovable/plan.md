
# Restaurante O Amoroso — Website

## Página Única (Index)

### 1. Header / Topo
- Nome "Restaurante O Amoroso" em tipografia elegante serifada
- Navegação simples: Início, Sobre, Galeria, Localização
- Fundo em tons de bege quente com estilo minimalista

### 2. Hero Section
- Grande área com emblema/logo decorativo de fundo (elemento gráfico subtil)
- Título principal com tagline acolhedora (ex: "Sabores da tradição portuguesa")
- Botão CTA para reservas ou ver menu

### 3. Sobre / Informação Inicial
- Breve descrição do restaurante, história e filosofia
- Destaque da cozinha tradicional portuguesa
- Ícones para especialidades (cozinha caseira, vinhos, ambiente familiar)

### 4. Galeria de Imagens
- Grid de imagens com placeholders elegantes
- Layout responsivo em grid 2-3 colunas

### 5. Localização
- Secção com mapa embed (Google Maps placeholder)
- Morada, horário de funcionamento, contactos

### 6. Rodapé
- Informação mínima do restaurante
- Link discreto: "Website por digitalsanity.pt"

### Design
- **Paleta**: Tons de bege (#F5F0E8, #E8DFD0, #D4C5A9), castanho escuro (#3C2A1A), branco creme
- **Tipografia**: Serifada elegante para títulos, sans-serif limpa para corpo
- **Estilo**: Minimalista, quente, acolhedor — tradicional português
